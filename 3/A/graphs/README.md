## Простые графы

### Теория

- [Фоксфорд](https://foxford.ru/wiki/informatika/teoriya-grafov)
- [Яндекс](https://education.yandex.ru/handbook/algorithms/article/priroda-grafa)

Пока надо только понимать, что такое граф, знать ряд базовых определений (вершина, ребро, путь и т.д.), и уметь хранить графы в программе(матрица смежности, списки смежных вершин). Из способов хранения графа пока будет достаточно матрицы смежности, про остальные можешь прочитать для сведения.
